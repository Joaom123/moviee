# Moviee
Exibição de informações sobre filmes.

## Informações
* API de utilizada: [OMDb API - The Open Movie Database](http://www.omdbapi.com/)
* Framework CSS: [Materialize](https://materializecss.com/)
* Library Frontend: [React](https://reactjs.org/)
